## dev

work in `src` folder

```bash
git clone https://github.com/bamdadsabbagh/boilerplate-static-site-webpack-sass-js
cd boilerplate-static-site-webpack-sass-js
yarn
yarn start
```

## build

outputs in `dist` folder

```bash
yarn build
```
